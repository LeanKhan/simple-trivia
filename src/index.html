Simple Trivia API

To get 10 random questions make a GET request to

https://trivia-123.herokuapp.com/get-questions

Body:

null

Response:

{ "questions": ArrayOfQuestions[], "length": Number }

--- These questions you generated will be saved in the database.json file ---

To answer those specific questions, make a POST to:

https://trivia-123.herokuapp.com/get-questions

Body:

{ "answers": ["A"..."B"] }

Response:

{"score_in_percentage": "70%","score_in_digits": 70}
